0.MySQL入门

2022/9/5 数据库 sql mysql mysql 数据库 sql

1.服务启动

启动MySQL服务的方式有命令行启动以及图形界面方式启动。

命令行启动如下：

MySQL服务启动:

net  start  mysql

MySQL服务停止:

net stop mysql

2.登录

启动MySQL 服务后通过如下的命令可以登录

mysql -u root -P 端口号  -h 主机 -p

其中-u 代表的是用户 其后的root代表root权限的用户。-P代表端口号。-h代表主机自己本地的主机就写llocalhost ,远程连接的主机就写主机的ip地址。-p代表密码。

输入以上命令后就可以输入密码登录了。

如果访问的是本机，且端口号为默认的3306，则-P和-h可以不写,改为如下的简写

mysql -u root -p

3.退出登录

输入quit或exit

4.查看编码命令

show variables like 'character_%'     /*查看字符集*/
show variables like 'collation_%'     /*查看校对规则*/

如果是mysql5则需要更改字符集为utf-8，如果是mysql8则默认为utf-8

  • 修改mysql数据目录（data）下的my.ini配置文件

  • [mysql]#大概在63行，在其下添加
...
default-character-set = utf8 #默认字符集
...
[mysql]#大概在76行，在其下添加
...
character-set-server=utf-8
collation-server=utf8_general_ci

  • 修改字符集后重启服务
