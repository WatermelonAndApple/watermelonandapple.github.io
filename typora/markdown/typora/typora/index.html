Typora的基本使用教程

2022/9/3 typora markdown typora

1.Typora简介

Typora是一款轻便简洁的Markdown编辑器,支持即使渲染技术。即时渲染使得写Markdown就像写word文档一样流畅自如，所见即所得。

Typora的优点如下：

  • 界面简洁美观
  • 可实时预览
  • 扩展语法
  • 具有夸平台的特性
  • 免费（重要）

Typora官网

2.Typora的使用

2.1文字处理

2.1.1标题

Typora的标题一共有六级,其是用“#”实现的，标题前面加一个“#”，代表一级标题，两个“#”代表二级标题，依次类推（注意符号和标题之间应该有空格，否则标题不生效）。

标题的快捷键一级标题: ctrl + 1 ; 二级标题快捷键: ctrl + 2,以此类推。

打开源代码模式以上标题的表示形式如下图所示

2.1.2强调

在强调的内容前后分别加入两个“*”号。

例：我是强调

2.1.3斜体

在前后分别加一个“*”号。

例:我是斜体

2.1.4删除线

在内容前后分别加上两个”~”号.

例:斜体

2.2列表处理

2.2.1有序列表

有序列表输入数字“1” + ”.“ + 空格,即可以自动开始有序列表。

例：

  1. 有序列表1
  2. 有序列表2
  3. 有序列表3

2.2.2无序列表

无序列表输入“*” + 空格 ，自动开始无序列表。

例：

  • 无序列表1
    • 无序列表1的子列表1
    • 无序列表1的子列表2
  • 无序列表2
    • 无序列表2的子列表
  • 无序列表3

2.2.3表格

table表格Ctrl + T，在弹出的对话框中选择行列数，自动生成列表。可以很方便的对表格进行编辑。

2.3.5代码

单行代码:在代码的两端都加上“`”符号(在Tab键的上方，英文输入法)。

print("hello world!")

代码块:输入三个“~”号,可以选择编程语言

#include<stdio.h>
int main(){
    printf("Hello  world!");
    return 0;
}
