SpringBoot整合Email

2023/1/8 SpringBoot SpringBoot

1.前期准备

在QQ邮箱中点击设置，点击账户选项，向下拉选择POP3/SMTP服务，如下图所示：

我们获取授权码后，就可以在SpringBoot工程中配置文件中进行配置了。

2.使用SpringBoot发送邮箱

2.1 环境搭建

引入Email依赖：

      <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-mail</artifactId>
        </dependency>

然后在SpringBoot配置文件中加入相关的配置信息

#QQ邮箱配置
spring:
  mail:
    host: smtp.xxx.com             	#这里换成自己的邮箱类型 例如qq邮箱就写smtp.qq.com
    username: xxxxx.@xx.com   		#发送邮件的邮箱地址
    password: xxxxxxxxxx   		 	#客户端授权码
    protocol: smtp               	 #发送邮件协议
    properties:
      mail:
          smtp.auth: true
          smtp.port: 465          	#端口号465或587
          smtp.starttls.enable: true
          smtp.starttls.required: true
          smtp.ssl.enable: true     	   #开启SSL
    default-encoding: utf-8
    from:  xxxxx.@xx.com  	            #发送邮件的地址和上面的username的邮箱地址一样

2.2 发送普通邮件

普通邮件是指最为普通的存文本邮件。创建MailService类用来封装邮件的发送，代码如下：

@Service
public class MailService {
    private final Logger logger= LoggerFactory.getLogger(this.getClass());
    /*SprinBoot提供的一个发送邮件的 简单抽象，使用的是下面的接口
    * ，这里只需要直接注入就可以直接使用了*/
    @Autowired
    private JavaMailSender mailSender;
    //配置文件中的自己的QQ邮箱
    @Value("${spring.mail.from}")
    private String from;   //邮件发送方
    /**
     * 发送普通的邮件
     * @param to   邮件接受方
     * @param subject    邮件主题
     * @param content    邮件内容
     */
    public void sendSimpleMail(String to,String subject,String content){
        //创建SimpleMailMessage邮件对象
        SimpleMailMessage message = new SimpleMailMessage();
        message.setFrom(from);  //邮件发送者
        message.setTo(to);     //邮件接受方
        message.setSubject(subject);     //邮件主题
        message.setText(content);   //邮件内容
        //发送邮件
        mailSender.send(message);
    }
}

简单邮件通过SimpleMailMessage对象封装打包数据，最后通过JavaMailSender类将数据发送出去。

2.3 HTML邮件

很多时候，需要邮件带有美观的样式。这时候，就可以使用HTML的样式。我们需要使用JavaMailSender的createMimeMessage方法，构建一个MimeMessage，然后使用MimeMessage实例创建出MimeMessageHelper。

在MailService类中添加如下代码如下：

    /**
     * 发送HTML邮件
     * @param to   邮件接受方
     * @param subject    邮件主题
     * @param content    邮件内容
     */
    public void sendHtmlMail(String to,String subject,String content){
        //获取MimeMessage对象
        MimeMessage message=mailSender.createMimeMessage();
        MimeMessageHelper messageHelper = null;
        try{
            messageHelper= new MimeMessageHelper(message,true);
            messageHelper.setFrom(from);   //邮件发送者
            messageHelper.setTo(to);        //邮件接收方
            messageHelper.setSubject(subject);   //邮件主题
            messageHelper.setText(content,true);   //邮件内容，HTML邮件
            mailSender.send(message);   //发送邮件
            logger.info("邮件已经发送");
        }catch (Exception e){
            logger.error("发送邮件时产生异常",e);
            e.printStackTrace();
        }
    }

配置完成后在SpringBoot的主测试类中进行测试，代码如下：

@SpringBootTest
class SpringBootEmailApplicationTests {
    @Autowired
    private  MailService mailService;
    //TemplateEngine对模板进行渲染
    @Autowired
    private TemplateEngine templateEngine;
    /**
     * 测试模板邮件
     */
    @Test
    void testTemplatesEmail(){
        //Thymeleaf模板传递值，并解析成字符串
        Context context =new Context();
        context.setVariable("id","001");
        String emailContext = templateEngine.process("emailTemplate",context);
        mailService.sendHtmlMail("3334059895@qq.com","主题：模板邮件",emailContext);
    }
    /**
     * 测试发送普通邮件
     */
    @Test
    void testSendEmail(){
        mailService.sendSimpleMail("3334059895@qq.com","主题：普通邮件","第一封有邮件");
    }
    /**
     * 测试Html邮件
     */
    @Test
    void testSendHtmlEmail(){
        mailService.sendHtmlMail("3334059895@qq.com","主题：HTML邮件","<h1>HTML邮件</h1><h2>h2标题</h2>");
    }

}
