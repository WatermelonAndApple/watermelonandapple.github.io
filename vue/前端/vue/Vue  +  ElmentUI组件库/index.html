Vue + ElementUI组件库

2022/5/15 vue vue 前端 ElementUI

Vue官网:https://cn.vuejs.org/

ElementUi官网：https://element.eleme.cn/#/zh-CN

1.ElmentUI的安装

npm安装（推荐）

使用npm的方式安装能够更好的和webpack打包工具配合使用。

npm i element-ui -s
引入外部文件的形式

通过cdn在页面上引入css文件和js文件

<!-- 引入样式 -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
<!-- 引入组件库 -->
<script src="https://unpkg.com/element-ui/lib/index.js"></script>

2.快速上手

安装后，需要将ElementUI引入，可以根据需要引入部分组件,也可以完整引入。

完整引入

在main.js文件中写入如下内容：

import Vue from 'vue';              //引入Vue
//引入ElementUI组件库
import ElementUI from 'element-ui';       
//引入ElementUI全部样式
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';
//应用ElementUI
Vue.use(ElementUI);

new Vue({
  el: '#app',
  render: h => h(App)
});
按需引入

借助 babel-plugin-component，我们可以只引入需要的组件，以达到减小项目体积的目的。

首先，安装 babel-plugin-component：

npm install babel-plugin-component -D

然后，将 .babelrc 修改(新版的vue生成的文件为babel.config.js)为：

{
  "presets": [["@babel/preset-env", { "modules": false }]],
  "plugins": [
    [
      "component",
      {
        "libraryName": "element-ui",
        "styleLibraryName": "theme-chalk"
      }
    ]
  ]
}

接下来，如果你只希望引入部分组件，比如 Button 和 Select，那么需要在 main.js 中写入以下内容：

import Vue from 'vue';
import { Button, Select } from 'element-ui';
import App from './App.vue';

Vue.component(Button.name, Button);
Vue.component(Select.name, Select);
/* 或写为
 * Vue.use(Button)
 * Vue.use(Select)
 */

new Vue({
  el: '#app',
  render: h => h(App)
});

可以在components.json中查看组件库有哪些组件。

3.全局配置

在引入 Element 时，可以传入一个全局配置对象。该对象目前支持 size 与 zIndex 字段。size 用于改变组件的默认尺寸，zIndex 设置弹框的初始 z-index（默认值：2000）。按照引入 Element 的方式，具体操作如下：

完整引入 Element：

import Vue from 'vue';
import Element from 'element-ui';
Vue.use(Element, { size: 'small', zIndex: 3000 });

按需引入 Element：

import Vue from 'vue';
import { Button } from 'element-ui';

Vue.prototype.$ELEMENT = { size: 'small', zIndex: 3000 };
Vue.use(Button);

按照以上设置，项目中所有拥有 size 属性的组件的默认尺寸均为 ‘small’，弹框的初始 z-index 为 3000。

至此，Vue + ElementUI的开发环境就已经搭建好了。各个组件的使用方法请参考官方文档。
